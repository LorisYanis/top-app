## Top App

Top app - topics application for every studying topic you just can imagine

# Set up project

1. Copy repository

```bash
git clone https://github.com/LorisYanis/top-app.git
```

2. navigate to top-app folder

```bash
cd top-app
```

3. install dependencies

```bash
npm i
```

4. start the project

```bash
npm run dev
```
